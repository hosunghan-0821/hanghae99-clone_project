## ****🍀 프로젝트 소개****

 저희 조는 이번 주차에 우아한 형제들에서 운영하고 있는 여러 가지 서비스 중에 문구 및 여러 생활용품들을 판매 및 배송을 하는 플랫폼인 배민 문방구를 클론 코딩 하였습니다.<br>
